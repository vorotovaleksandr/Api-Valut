# testReact
## Requirements for use

+ installed node
  + https://nodejs.org/en/download/
+ installed mongoDB
  + https://docs.mongodb.com/manual/installation/
### Super Easy Install
```
git clone https://github.com/vorotovaleksandr/testReact.git
cd test react 
cd client
yarn install
cd ..
yarn install
yarn dev
```
##### Open your browser 
Your browser opened http://localhost:3000/
